# Мои проекты

## **res-space barmanager (BAR_V3)**  
Комплексная система управления баром для автоматизации бизнес-процессов: управление инвентарём, планирование заготовок, заказы поставщикам и аналитика.

### Технологический стек
- **Бэкенд:** ASP.NET Core WebAPI
- **Фронтенд:** React.js + TypeScript
- **База данных:** PostgreSQL
- **Кэширование:** Redis
- **Фоновые задачи:** Hangfire
- **Контейнеризация:** Docker

### Функциональные модули
- Аутентификация и авторизация с ролевой системой (Бармен, Старший бармен, Менеджер, Администратор)
- Управление продуктами, категориями, единицами измерения, штрих-кодами, минимальными остатками
- Инвентаризация с историей изменений и фиксацией пользователя
- Заказы поставщикам с генерацией PDF
- Планирование заготовок и прогнозирование на основе исторических данных
- Аналитика и отчёты, экспорт данных

### Структура проекта (основные директории)

- **Backend (C#):**
  - `Controllers/` - API контроллеры
  - `Services/` - Бизнес-логика
  - `Models/` - DTO и модели
  - `Data/` - Работа с базой данных
  - `Infrastructure/` - Инфраструктурные компоненты

- **Frontend (React):**
  - `src/components/` - Компоненты интерфейса
  - `src/pages/` - Страницы приложения
  - `src/services/` - API клиенты
  - `src/utils/` - Вспомогательные функции

### Достижения
- Полностью автоматизирован процесс управления инвентарём
- Сокращение времени на планирование заготовок на 60%
- Улучшение контроля за сроками годности продуктов
- Удобный интерфейс для всех категорий пользователей

### Технические особенности
- RESTful API с JWT аутентификацией
- Реализация ролевой модели доступа
- Интеграция с PDF генератором для отчётов
- Кэширование часто используемых данных
- Фоновые задачи для обработки больших объёмов данных
